# AI-Risk-Program-UAE-Europe
Industry-neutral AI Risk Program with two sub-projects (UAE and Europe). End-to-end playbook covering governance, policies, risk taxonomy, assessment methodology, tooling, vendor management, controls, metrics and a phased implementation roadmap. Aligned to EU AI Act, NIST AI RMF, ISO/IEC 42001, UAE AI Office, CBUAE, DOH Abu Dhabi and DIFC.
